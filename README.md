# 525 — Five Two Five · Specialty Coffee — موقع الكوفي

نموذج تفاعلي لموقع كوفي **525** بفكرة **الفيديو الحيّ في الخلفية** الذي لا يتوقّف:
تفتح الصفحة على واجهة الكوفي → «تدخل» للكوفي → تشوف الأجواء والبارستا،
وكل قسم من المنيو تضغطه يغيّر الفيديو في الخلفية لِما يناسبه.

---

## التشغيل

الفيديوهات ما تشتغل بفتح `index.html` مباشرةً (قيود المتصفّح على الملفات المحلية).
شغّل سيرفر محلي من داخل مجلد `site`:

```bash
python -m http.server 8712      # أو:  npx serve -l 8712
```
ثم افتح: **http://127.0.0.1:8712**

**روابط مباشرة (تفتح على الحالة مباشرة — تنفع للمشاركة):**
`#hot` · `#cold` · `#v60` · `#pastry` · `#mood` · `#search` · `#cart`
مثال: `http://127.0.0.1:8712/index.html#pastry`

## المزايا الحالية

- **خلفية فيديو دائمة** بطبقتين + انتقال **Cross-fade** ناعم (ما تتوقّف أبداً).
- **دخول سينمائي**: زر «ادخل الكوفي» يعمل انتقال «دخول من الباب» (تكبير + بلور) ثم يبدّل للأجواء.
- **المنيو الرسمي كامل بالأسعار (M/L)**: أربعة أقسام (الساخن / التقطير / البارد / الحلى) وكل قسم
  مقسّم لمجموعات فرعية — الساخن (حار فقط) · مختصّة V60 · البارد (بارد، ماتشا، سموذي، موهيتو، فرابتشينو، عصائر) · الحلى (تشيزكيك، كوكيز).
- **أيقونات line-art بستايل هويّتكم**: كل صنف عليه أيقونة مرسومة (كوب/كأس بارد/دريبر/ماتشا/عصير/كيك)،
  وخلف عنوان كل قسم رسمة line-art كبيرة خفيفة (زخرفة العلامة).
- **بطاقات أصناف بصور حقيقية**: قسم الحلى يعرض صور المنتجات الفعلية
  (تشيزكيك جيري / لوتس / بستاشيو / أوريو + كوكيز). لاحقًا نستبدل أيقونات المشروبات بصور كل صنف.
- **زر الكوكي «good mood»** (من تصميمكم الأصلي «stress off / good mood on»):
  سويتّه عنصر تفاعلي حقيقي — لمّا تشغّله يدفّئ ألوان المشهد ويشغّل مقطع التوقيع (الفلامبيه)
  ويطلع تنبيه لطيف. (فوق يسار)
- **أزرار تواصل حقيقية**: إنستغرام / واتساب / خرائط جوجل / اتصال (تحت يسار).
- **صوت** قابل للتشغيل (مكتوم افتراضياً)، تصميم **RTL** كامل، وخطوط أنيقة.

### مزايا التطوير الأخير
- **بحث فوري + فلاتر** (🔍 فوق): يبحث بكل الأصناف (عربي/إنكليزي) مع فلاتر (الكل · الأكثر طلباً · جديد · بدون حليب)، وتقدر تضيف للسلة مباشرة من النتائج.
- **سلة تُحفظ على الجهاز** (🛒 فوق) — بدون واتساب: أضف أصناف (M/L)، عدّل الكميات، والمجموع يتحدّث. **تُحفظ تلقائياً** (localStorage) فتقدر تجهّزها من البيت أو يجمّعها شخص للكروب، وزر **«نسخ الطلب»** ينسخ الطلب نصّاً لترسله بأي وسيلة.
- **شارات** على الأصناف: الأكثر طلباً · جديد · موسمي.
- **زر «مشاركة»** على كل صنف: يولّد **بطاقة/ستوري** جاهزة (صورة + السعر + لوغو 525) للتنزيل أو المشاركة.
- **«اقترح لي 🎲»**: يختار صنف عشوائي، يفتح قسمه ويميّزه.
- **تنقّل سينمائي**: نقاط جانبية (يمين) + **التمرير بعجلة الماوس** يتنقّل بين الأقسام.
- **تصنيفات بارزة**: شرائح تنقّل أعلى كل قسم تقفز للمجموعة وتبدّل مشهدها (كل تصنيف جاهز لفيديو خاص لاحقاً).
- **وضع نهاري/ليلي** يتبع ساعات الدوام (٧ص–١ص): يبدّل التدرّج اللوني ويعرض حالة **مفتوح/مغلق**.
- **لمسات سينمائية**: حبيبات فيلم أقوى + تدرّج لوني + **مؤشّر ماوس حبّة قهوة** + بخار line-art متحرّك فوق أيقونة القسم.
- **متجاوب للجوال**: اللوحة بعرض كامل، شريط الأقسام أفقي، تأثيرات أخفّ.

## تعديل المحتوى (كله في مكان واحد)

افتح `assets/js/app.js`:

### 1) تبديل فيديو أي مشهد → كائن `SCENES`
```js
var SCENES = {
  exterior: 'assets/video/intro-exterior.mp4',
  interior: 'assets/video/welcome.mp4',
  hot:      'assets/video/hot.mp4',
  cold:     'assets/video/cold-matcha.mp4',
  v60:      'assets/video/hot-alt.mp4',
  pastry:   'assets/video/pastry-cookies.mp4',
  signature:'assets/video/signature.mp4'   // مقطع الـ good mood
};
```

### 2) تعديل أصناف/أسعار/صور المنيو → مصفوفة `SECTIONS`
كل قسم فيه `groups` (مجموعات فرعية)، وكل صنف يُكتب بالدالة المختصرة:
```js
it('الاسم بالعربي','NAME EN', سعر_M, سعر_L, 'نوع_الأيقونة_أو_مسار_صورة')
// أمثلة:
it('كافي لاتيه','CAFE LATTE', 4000, 5000, 'hot')                 // أيقونة line-art
it('كوكيز','COOKIES', 0, 0, 'assets/img/menu/cookie.jpg')        // صورة حقيقية
// السعر 0 = لا يظهر سعر · أنواع الأيقونات: hot · iced · drip · matcha · juice · cake
```

> الصور المصغّرة للمنتجات في `assets/img/menu/`، وبوسترات الفيديو في `assets/img/posters/`.

## أفكار للتوسّع (المرحلة القادمة)

1. **الأسعار** لكل صنف (تُركت فارغة حتى تُعتمد منكم) + شارة «الأكثر طلباً».
2. **صفحة/قسم الأصناف الموسمية** (ثيم شتوي — عندنا مقطع الفلامبيه والأكواب الشتوية).
3. **زر «اطلب الآن»** يفتح واتساب برسالة جاهزة بالطلب.
4. **تأثير الـ line-art** بهويّتكم (رسمة الوجه/الكوكي) كعناصر زخرفية وشعارات القسم.
5. **صوت أجواء** خفيف + خيار كتم.
6. **نسخة أخفّ للجوال** (ضغط الفيديوهات + تحميل كسول) لتحسين السرعة على الشبكات.
7. **لوحة تحكّم بسيطة** لصاحب الكوفي لتبديل المقاطع والأصناف بدون كود.
